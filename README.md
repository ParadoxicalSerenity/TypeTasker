# TypeTasker
[![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](https://choosealicense.com/licenses/mit/)

Typescript first task runner.

## Installation

Install typetasker as a development dependency with npm

```bash
  npm i typetasker -D
```
